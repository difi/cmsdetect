
---------------------------
WHAT IS CMSDETECT?
---------------------------

CMSDetect is an application designed to check if a given website is using a content managing system, and which CMS it most likely is.
In it's current state it's checking for these Content Managing Systems and features:

CMS:
    - Contao
    - Django
    - Drupal
    - Joomla
    - Modx
    - Radiant
    - SilverStripe
    - Wordpress
    - Zope

Features:
    - AddThis
    - AngularJs
    - Bootstrap
    - Javascript
    - JQuery
    - Stylesheet

---------------------------
REQUIREMENTS
---------------------------

Maven
Spring
Tomcat (or similar)

---------------------------
HOW TO RUN
---------------------------

To run cmsdetect-cli:
mvn clean tomcat7:run -pl :cmsdetect-web -am
